L'Homebrew Launcher ha numerosi entrypoint, o metodi per essere avviato.
{: .notice}

Soundhax is compatible with versions 9.0.0 through 11.2.0 in the EUR, JPN, KOR, and USA regions.
{: .notice--info}

#### Cosa serve

+ L'[Homebrew Starter Kit](http://smealum.github.io/ninjhax2/starter.zip)
+ The latest release of [Soundhax](http://soundhax.com/) *(for your device and region)*
+ Il file [otherapp payload](https://smealum.github.io/3ds/#otherapp) *(adatto alla versione e alla regione della console)*

#### Istruzioni

1. Copia il _contenuto_ della cartella `starter` dell'archivio `starter.zip` nella root della tua scheda SD
2. Copy the Soundhax `.m4a` to the root of your SD card
3. Copia il file otherapp payload nella root della tua scheda SD e rinominalo in `otherapp.bin`
4. Reinserisci la scheda SD nel 3DS
5. Avvia Sound Nintendo 3DS
6. Scorri tutti i consigli proposti, chiudi l'applicazione normalmente, poi riaprila
  + Launching Soundhax immediately would cause these tips to appear on every launch of the Nintendo 3DS Sound until this is done
7. Seleziona `/Scheda SD`, quindi avvia "<3 nedwill 2016"

  + Se la console dovesse bloccarsi, forzane lo spegnimento tenendo premuto il pulsante di accensione, quindi riprova
8. La console dovrebbe ora avviare l'Homebrew Launcher
